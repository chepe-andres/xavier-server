# Xavier-Server




My personal home server built on Centos-stream 10 and Bootc, includes ZFS, Nvidia drivers and other utilities I use for self-hosting! 

** This is not meant for use outside of my own personal use, mileage may vary if you try to use it! **

This repository was made from ublue's template for building your own custom [bootc](https://github.com/bootc-dev/bootc) image. This template is the recommended way to make customizations to any image published by the Universal Blue Project.
